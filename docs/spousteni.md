# Jak projekt spustit?

To je jednoduché!

Stačí pouze zadat do konzole

```terminal
npm run server.js
```
a poté si ve vebovém prohlížeči otevřít [localhost:3000](localhost:3000)

::: tip
Pokud chcete spustit dokumentaci, tak použijte 
```terminal
npm run docs:dev
```
:::
